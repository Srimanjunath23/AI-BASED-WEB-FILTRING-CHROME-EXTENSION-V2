How to Use

Click the SafeGuard icon in your browser toolbar to access:

  • Filter toggles for different content types
  • Sensitivity adjustment slider
  • Educational mode toggle
  • Password protection settings

When content is blocked:

  • A block page will appear with explanation
  • You can click "Go Back" to return
  • Admin users can override with password

Click "Show Image" and enter admin password to view blurred images
